# Grafika Projekt - Minesweeper 3D

Projekt został wykonany na szkielecie pobranym ze strony przedmiotu Grafika Komputerowa i Wizualizacja

## Controls
* Arrows -> movement of chosen tile
* WASD -> Camera movement
* O, P -> Bring the camer closer / further from the board
* Space -> Reveal a tile
* E -> Toggle the flagging of chosen tile
* R -> Reset the game
* ESC -> Close the game

## Features
* Camera movement
* Random bomb placement
* Functional bombs
* Functional cursor
* Flagging mechanic
* Shading
