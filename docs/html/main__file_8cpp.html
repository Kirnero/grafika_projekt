Detailed Description

Main logic and rendering center for the 3D Minesweeper project.

Implements the game loop, input handling, camera control, board logic, and shading for a 3D Minesweeper scene.

Features include random mine placement, cursor navigation, flag toggling, reveal animation, light shading, and textured explosion rendering.

Note
Controls:
  • Arrow keys: move selected tile
  • W/A/S/D: rotate camera
  • O/P: zoom in/out
  • Space: reveal selected tile
  • E: toggle flag
  • R: reset game
  • Esc: close game
